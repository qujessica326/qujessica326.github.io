Definitions

Serif
A typeface style with small decorative strokes at the ends of letters. This site uses Georgia, a serif font, with a serif fallback to ensure readable, elegant text.
Padding
The space inside a box around its content. It adds breathing room between text or images and the box’s border. In this site, padding is used on the <main> area to create comfortable spacing.
Alt text
Short for “alternative text.” It describes the content or purpose of an image for screen readers and appears when images fail to load, improving accessibility for all users.
Margin
The space outside a box that separates it from other elements on the page. Margins help structure the layout and prevent elements from crowding each other.
Class Selector
A CSS selector that starts with a period (.). It is used to apply the same style to multiple elements that share the same class name.
