# M1 Website Starter

This folder contains a 5–page, interlinked, accessible site that meets the LIS 351 M1 rubric items.
Edit the page content and replace the SVG placeholder images with your own images in `/images`.
Then upload the whole folder to GitHub Pages.

## Files
- index.html (home)
- about.html
- art.html  (3 images + figure/figcaption)
- experience.html (unique two–column layout via `.split`)
- favorites.html (ordered list + description list)
- styles.css (external stylesheet)

## Rubric Checklist (where satisfied)
- `<!DOCTYPE html>` and `<title>` on all pages ✓
- Headings (h1/h2/h3) + paragraphs ✓
- Styling changes ≥ 4 elements (bold, borders, font variant, etc.) ✓
- List (ul/ol) ✓ and Description List (dl/dt/dd) ✓
- Three images (art.html uses self-created SVG placeholders) ✓
- One external hyperlink with meaningful text (several present) ✓
- All images have `alt` text ✓
- `<nav>` with links to all pages on each page ✓
- External CSS: non-default serif font + background color + spacing ✓
- Unique layout: `experience.html` uses `.split` two-column grid ✓

## Validate
Open each page and run it through https://wave.webaim.org/ (no Errors; Alerts are OK per rubric).
